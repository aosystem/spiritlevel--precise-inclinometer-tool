# Spirit Level: Precise Inclinometer 📏📐

A reliable and minimalist **Spirit Level** (Bubble Level) built with **Flutter**. This utility turns your smartphone into a precise measurement tool, allowing you to accurately determine the angle of inclination for any surface or object.

## 🌟 Key Features
- **High-Precision Measurement:** Leverages internal accelerometer and gyroscope sensors for real-time angle detection.
- **Horizontal & Vertical Support:** Designed to measure tilt accurately by placing the device against an object.
- **Minimalist Interface:** A clean, distraction-free UI that focuses entirely on data accuracy.
- **Real-time Feedback:** Instantly displays the degree of inclination with smooth visual updates.
- **Global Support:** Fully localized in **50+ languages**, making it a versatile tool for international users.

## 💡 Best For...
- **DIY & Home Improvement:** Ensuring shelves, picture frames, and furniture are perfectly level.
- **Construction & Carpentry:** Quick on-site verification of angles and slopes.
- **Photography & Videography:** Helping to align tripods and cameras for the perfect shot.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Direct integration with the device's hardware sensors (accelerometer) combined with noise-filtering logic for stable readings.

*Note: This repository contains the core logic and source code. Assets (images, icons, and localization files) are excluded to protect intellectual property.*
